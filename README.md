# Zoe Dashboard

Sistema de Gestão Escolar com autenticação completa.

## 🚀 Instalação

```bash
npm install
npm run dev
```

## 🔐 Configuração

1. **Crie a tabela `usuarios` no Supabase:**
```sql
create table public.usuarios (
  id bigint generated by default as identity not null,
  nome text not null,
  foto_url text null,
  email text null,
  senha_hash text null,
  data_criacao timestamp with time zone null default now(),
  ativo boolean null default true,
  constraint usuarios_pkey primary key (id),
  constraint usuarios_email_key unique (email)
);

create index IF not exists idx_usuarios_email on public.usuarios using btree (email);
create index IF not exists idx_usuarios_ativo on public.usuarios using btree (ativo);

-- Desabilitar RLS temporariamente
ALTER TABLE public.usuarios DISABLE ROW LEVEL SECURITY;
```

2. **Configure as variáveis de ambiente:**
```env
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=sua-chave-secreta
NEXT_PUBLIC_SUPABASE_URL=sua-url-supabase
NEXT_PUBLIC_SUPABASE_KEY=sua-chave-supabase
```
